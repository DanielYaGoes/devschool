# Domínio

## O que é o DevSchool?

DevSchool é uma plataforma de estudos onde professores podem criar e organizar cursos, aulas, disciplinas e listas de questões, enquanto alunos podem acessar os conteúdos disponíveis, acompanhar seus estudos e exercitar o aprendizado.

O sistema também terá uma área administrativa/interna para gerenciamento de usuários, cursos e permissões de acesso.

## Atores

### Administrador

Usuário interno responsável por gerenciar o sistema, usuários, cursos e permissões.

### Professor

Usuário interno responsável por criar cursos, aulas, disciplinas e questões.

### Aluno

Usuário externo que acessa os cursos disponíveis, assiste aulas e realiza listas de questões.

## Módulos

- Autenticação
- Usuários
- Cursos
- Disciplinas
- Aulas
- Matrículas
- Questões
- Listas de Questões
- Relatórios
- AulaXDisciplina
- DisciplinaXCurso

## MVP - Versão 1

Na primeira versão, o sistema deverá permitir:

- cadastrar usuários internos;
- cadastrar alunos;
- cadastrar cursos;
- cadastrar disciplinas;
- cadastrar aulas;
- vincular aulas a disciplinas;
- vincular disciplinas a cursos;
- matricular alunos em cursos;
- permitir que o aluno visualize seus cursos e aulas.
