# DevSchool

> Projeto criado para estudar Engenharia de Software na prática.

## Objetivo

O DevSchool é uma plataforma de estudos desenvolvida como projeto de longo prazo.

O objetivo não é apenas criar funcionalidades, mas aprender conceitos fundamentais de desenvolvimento profissional, incluindo:

- Arquitetura de Software
- Clean Code
- SOLID
- Spring Boot
- PostgreSQL
- Docker
- Git
- Testes
- Documentação
- Front-end
- Segurança
- Cache
- Microsserviços (futuramente)

---

## Stack

Backend

- Java 21
- Spring Boot
- Maven

Banco

- PostgreSQL

Ferramentas

- Git
- Flyway
